┌──────────────────────────────────────────────────────────────────┐
│ SEARCH & DISCOVERY                                               │
├──────────────────────────────────────────────────────────────────┤
│ <space>rs    Semantic Search           Natural language query    │
│ <space>rw    Search Word                Word under cursor        │
│ <space>rf    Find Functions             By description           │
│ <space>rm    Find Modules               By description           │
└──────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────┐
│ CODE NAVIGATION                                                  │
├──────────────────────────────────────────────────────────────────┤
│ <space>rc    Find Callers               Who calls this function  │
│ <space>rg    Graph Stats                Knowledge graph info     │
└──────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────┐
│ REFACTORING (Project-Wide)                                       │
├──────────────────────────────────────────────────────────────────┤
│ <space>rr    Rename Function            Safe, atomic rename      │
│ <space>rR    Rename Module              Updates all references   │
└──────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────┐
│ ANALYSIS & SETUP                                                 │
├──────────────────────────────────────────────────────────────────┤
│ <space>ra    Analyze File               Current file only        │
│ <space>rd    Analyze Directory          Entire project           │
│ <space>rW    Watch Directory            Auto-reindex on change   │
└──────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────┐
│ ALTERNATIVE COMMANDS                                             │
├──────────────────────────────────────────────────────────────────┤
│ :RagexSearch          Same as <space>rs                          │
│ :RagexFunctions       Same as <space>rf                          │
│ :RagexModules         Same as <space>rm                          │
│ :RagexSearchWord      Same as <space>rw                          │
└──────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────┐
│ TYPICAL WORKFLOW                                                 │
├──────────────────────────────────────────────────────────────────┤
│ 1. Start Ragex:       cd ~/Proyectos/Ammotion/ragex &&          │
│                       mix run --no-halt &                        │
│                                                                  │
│ 2. First time:        <space>rd  (analyze project)              │
│                                                                  │
│ 3. Daily use:         Files auto-analyzed on save               │
│                       <space>rs  (search by description)        │
│                       <space>rc  (find callers)                 │
│                       <space>rr  (rename safely)                │
└──────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────┐
│ TELESCOPE NAVIGATION (in search results)                        │

│ Ctrl-n/p     Next/Previous result (alternative)                  │
└──────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────┐
